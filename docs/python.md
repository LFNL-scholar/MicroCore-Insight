# python 相关

## 1. 注释中关键字高亮
### 1. **TODO**（待办事项）
```python
# TODO: 这里需要优化代码，提高性能
```
> 许多 IDE 会在侧边栏高亮 TODO，并允许你快速导航到这些标记的位置。

### 2. **FIXME**（需要修复的错误）
```python
# FIXME: 这个函数在边界情况下会崩溃，需要修复
```
> 用于标记代码中存在问题的部分，提醒自己或团队成员修复。

### 3. **BUG**（已知的 bug）
```python
# BUG: 这里在 Python 3.8 以上可能会出现兼容性问题
```
> 用于标记已知的 bug，方便日后修复。

### 4. **HACK**（临时解决方案）
```python
# HACK: 目前使用了一个临时解决方案，需要重构
```
> 说明代码中存在临时的、非最佳的实现，可能需要重构。

### 5. **NOTE**（重要提示）
```python
# NOTE: 这里的参数需要传入字符串类型，否则会报错
```
> 记录重要的代码行为或逻辑，方便后续维护。

### 6. **XXX**（需要特别关注的代码）
```python
# XXX: 这段代码有点混乱，后续需要优化
```
> 用于标记不清晰、需要特别关注的代码部分。
